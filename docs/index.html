Before you begin

This is an Android class. To complete the coding exercises, you will need an Android development environment with either an emulator or a device to run the exercise on. Please see the Windows setup or macOS setup pages if you need help installing the Xamarin development environment.

Overview

This course covers Toolbar and app bar. You will see how to create a Toolbar, populate it from an XML menu file, and respond to item-click events. You will also integrate a Toolbar into an Activity by setting it as the Activity's app bar.

Objectives

  1. Add a Toolbar to an Activity
  2. Populate Toolbar Actions from an XML menu file
  3. Use a Toolbar as an Activity's app bar
  4. Set a navigation icon on an app bar

Exercises

Exercise materials are available from GitHub. The code can be cloned through a desktop Git client or downloaded directly as a ZIP file.

  1. Add a Toolbar to an Activity
    Exercise
    We begin by adding a Toolbar to an existing Activity and setting its title and logo.
  2. Populate Toolbar Actions from an XML menu file
    Exercise
    Here we use an XML menu file to populate the entries in a Toolbar's Actions menu.
  3. Add a checkable submenu
    Exercise
    Next we add a sub-menu filled with radio buttons.
  4. Use a Toolbar as an Activity's app bar
    Exercise
    This exercise integrates a Toolbar into an Activity by setting it as the Activity's app bar. This ensures it has standard positioning and uses the standard coding pattern for inflation and selection.
  5. Set a navigation icon on an app bar
    Exercise
    Finally, we add a navigation icon into the standard app-bar location and navigate the app in response to user taps.
